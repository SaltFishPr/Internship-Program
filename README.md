## 网络请求的问题

data包中UserInfo保存了用户所有信息（由于登陆的问题，默认为phone为1的用户采用get请求），MyValues保存了服务器的ip地址和端口在本地测试的时候请修改这个值为你flask框架所在主机的ip地址  
HomeFragment与EditActivity 中的GetInfoTask方法中构建的Url 后续的1应该修改为UserInfo.phone

## 项目简介

抖音（仿）是一款社交类的软件，通过抖音短视频 App 你可以分享你的生活，同时也可以在这里认识到更多朋友，了解各种奇闻趣事。
